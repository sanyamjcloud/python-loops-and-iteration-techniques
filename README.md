# Python Loops and Iteration Techniques

## Overview

This repository demonstrates the use of loops in Python through multiple practical scripts covering iteration, control flow, and real-world use cases.

## Topics Covered

* for loops
* while loops
* break and continue
* string iteration
* range with steps
* conditional logic inside loops

## Scripts Included

* `task2_for_loop.py` — Basic iteration using for loop
* `task3_while_countdown.py` — Countdown using while loop
* `task4_break_continue.py` — Control flow using break and continue
* `task5_iterate_string.py` — Iterating through strings
* `task6_multiplication_table.py` — Generate multiplication table
* `task7_range_with_steps.py` — Using range with step values
* `task8_loop_with_conditions.py` — Conditional logic inside loops
* `task9_real_world_example.py` — Practical use case of loops

## Usage

Run any script individually:

```bash
python <script_name>.py
```

## Learning Outcomes

* Understanding iteration patterns
* Applying loop control statements
* Writing efficient repetitive logic
